Reference notebooks
===================

Two notebooks are published as rendered reference pages. They are data-backed
reference material rather than narrative tutorials, so they sit apart from the
main learning path. The tutorials themselves live in the five sections:
:doc:`/concepts/index`, :doc:`/working-with-infomap/index`,
:doc:`/flow-models/index`, :doc:`/workflows/index`, and
:doc:`/robustness/index`.

- :doc:`options-guide`: what every Infomap option is for, with runnable
  examples and a complete reference table generated from the option metadata.
- :doc:`benchmark-performance`: estimate run time and memory from network size,
  threads, trials, and hierarchy depth, with empirical scaling laws.

For the survey-article companion notebooks, see
:doc:`/article-companion/index`.
